Instalar aplicación de React
1. En la terminal ejecutar el comando "npx create-react-app react-router"
2. Después, "cd react-router"
3. Por ultimo, ejecutar "npm install react-router-dom"
4. Ingresar los archivos y las carpetas acá
5. Instalar el Framework de CSS: "npm install bootstrap"
6. Ejecutar el código "npm start"
