Tutorial de Instalación de Windows en Notebooks

1. Preparación

Asegurate de tener los siguientes materiales:

  • Especificaciones mínimas de hardware de la notebook:
    • Procesador: 1 GHz o más rápido.
    • RAM: 1 GB para 32 bits o 2 GB para 64 bits.
    • Espacio en disco duro: 16 GB para 32 bits o 20 GB para 64 bits.
    • Tarjeta gráfica: DirectX 9 o posterior con controlador WDDM 1.0.
    • Pantalla: 800x600.
  • Dispositivos de entrada: Teclado y mouse.
  • Imágen ISO de Windows: Descargar desde la página oficial de Microsoft.
  • Una unidad USB: Capacidad mínima de 8GB.
  • Programa Rufus: Descargar Rufus.

2. Creación del USB Booteable

Seguí estos pasos para crear un USB booteable usando Rufus:

  • Conectá la unidad USB a tu computadora.
  • Abrí Rufus y selecciona la unidad USB.
  • En "Selección de arranque", elegí la imágen ISO de Windows.
  • Hacé clic en "Empezar" y esperá a que el proceso finalice.

3. Instalación de Windows

Para instalar Windows en tu notebook, seguí estos pasos:

  • Conectá el USB booteable a la notebook.
  • Encendé la notebook y apretá la tecla de acceso al menú de arranque (generalmente F2, F12 o Esc).
  • Seleccioná la unidad USB como dispositivo de arranque.
  • Seguí las instrucciones en pantalla para instalar Windows.

4. Finalización

Una vez completada la instalación, seguí las instrucciones en pantalla para configurar tu nuevo sistema operativo Windows.
